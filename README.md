# LibSkript

LibSkript je webová aplikace pro studenty – osobní studijní knihovna s AI,
která rozumí tvým materiálům.

## Vize
- Upload studijních materiálů
- Automatická AI kategorizace
- AI chat nad vlastními dokumenty
- Shrnutí, kartičky, kvízy

## Tech stack
- Next.js 14
- TypeScript
- Tailwind CSS
- AI (RAG architecture)
